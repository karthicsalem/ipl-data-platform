# ipl-data-platform

Learn Data Engineering and Data Analysis with IPL data

## About

- A data engineering/analytics project that aimed at learning, making use of data from a familiar topic(cricket) to build a portfolio project.
- This project is being built with python,SQL, and AWS at its core.
- Scope to be extended to implement orchestration using git actions, Airflow and analytics using PowerBI/Tableau.
- Basic Architecture Blueprint --> Raw JSON → Bronze → Silver → Gold
